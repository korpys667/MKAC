## Установка

Оффициальные билды доступны тут [GitHub Releases](https://github.com/korpys667/MKAC/releases). 

## Скомпилируйте сурсы

1. `git clone https://github.com/korpys667/MKAC.git`
2. `cd MKAC`
3. `./gradlew build`
4. Готовый .jar будет в папке `build/libs`
